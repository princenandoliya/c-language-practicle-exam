# C Practice Programs

This repository contains a collection of basic **C programming practice programs**. These programs are useful for beginners to understand fundamental concepts such as **switch-case**, **arrays**, **strings**, **pointers**, **functions**, and **pattern printing**.

Each program is written in a simple and clear way and compiled using **GCC**.
